## Quick reference

- **Sign in:** `/admin` on the current staging origin. Credentials are
  the backend `ADMIN_USERNAME` / `ADMIN_PASSWORD`. Sessions are held in
  browser session storage and cleared on sign-out.
- **Employees:** `/admin/employees` — review, approve, reject and export
  submitted inductions.
- **Invitations:** `/admin/invitations` — view issued access codes,
  delivery status and used/not-used state.
- **CSV export:** the Export CSV button on the Employees page produces a
  UTF-8 CSV with the same column order as the legacy spreadsheet.
